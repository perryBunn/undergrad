   Kinect Pong
   ===========
   
This scenario is part of the book 

    "Introduction to Programming with Greenfoot"
    (Second edition)
    by Michael Kölling
    ISBN: 0134054296

This scenario is used in chapter 12.

This scenario connects to the Microsoft Kinect sensor board (you need to
have one connected to run this scenario) and then lets you play a simple
'pong'-style game.
